# Servicios de IA Gratuitos para Reemplazar OpenAI

## Servicios Recomendados (Con API Key Gratuita)

### 1. **Groq** ⭐ (MÁS RECOMENDADO)
- **URL**: https://groq.com/
- **Plan gratuito**: Sí, muy generoso
- **Límites**:
  - Llama 3.3 70B: 14,400 requests/día, 64,000 tokens/minuto
  - Llama 3.1 8B: 14,400 requests/día, 60,000 tokens/minuto
  - Qwen 3 32B: 1,000 requests/día, 6,000 tokens/minuto
- **Velocidad**: Extremadamente rápido (LPU inference)
- **Modelos**: Llama, Qwen, Mistral, Gemma
- **Requiere**: Email (no tarjeta de crédito)
- **Compatible con**: OpenAI SDK (solo cambiar base URL)

### 2. **Google AI Studio** ⭐
- **URL**: https://aistudio.google.com/
- **Plan gratuito**: Sí
- **Límites**:
  - Gemini 2.5 Flash: 250,000 tokens/minuto, 250 requests/día
  - Gemini 2.0 Flash: 1,000,000 tokens/minuto, 200 requests/día
- **Modelos**: Gemini 2.5 Pro, Gemini Flash, Gemma
- **Requiere**: Cuenta de Google
- **Nota**: Los datos NO se usan para entrenamiento en UK/CH/EEA/EU

### 3. **Hugging Face Inference API**
- **URL**: https://huggingface.co/inference-api
- **Plan gratuito**: $0.10/mes en créditos
- **Límites**: Rate limits por modelo
- **Modelos**: Miles de modelos open source
- **Requiere**: Email

### 4. **Cohere**
- **URL**: https://cohere.com/
- **Plan gratuito**: Trial API keys
- **Límites**: Compartidos entre modelos
- **Modelos**: Command R, Command R+, Aya
- **Requiere**: Email

### 5. **Cerebras**
- **URL**: https://cerebras.ai/
- **Plan gratuito**: $5/mes en créditos
- **Límites**: 
  - Llama 3.3 70B: 14,400 requests/día
  - Qwen 3 235B: 14,400 requests/día
- **Velocidad**: Muy rápido

### 6. **GitHub Models**
- **URL**: https://github.com/marketplace/models
- **Plan gratuito**: Sí (límites restrictivos)
- **Modelos**: GPT-4.1, Llama, Mistral, DeepSeek
- **Requiere**: Cuenta de GitHub
